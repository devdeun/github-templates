# Issues Template 적용하기

## Issue forms로 멋진 Issues template 만들기

- `.github/ISSUE_TEMPLATE` 폴더 생성
- 해당 폴더 내부에 `yml` 파일 생성
  - 올바르게 작성했다면 별도의 설정 없이 `issue template`으로 설정됨
